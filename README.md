## Know Your City
A GeoGuessr-style web game where you guess locations based on real-world images from your city or region.

Players are shown 3 random images from a selected area (via the Mapillary API), then guess the location on a map. Supports both singleplayer and multiplayer modes.

#### [Play Now!](https://tiziangth.github.io/know-your-city/)
---

## Features
* Singleplayer and multiplayer gameplay
* 3 random street-level images per round
* Custom area selection on an interactive map
* Scoring based on distance between guess and actual location
* Real-time multiplayer with shared rounds and a leaderboard

---

## Tech Stack
* React (Frontend)
* Node.js + Express (Backend)
* Socket.io (Multiplayer)
* Mapillary API (Fetching images)
